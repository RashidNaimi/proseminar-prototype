# Einrichtung

1. **Node.JS installieren**:
In windows durch `winget install NodeJS.NodeJS`, in Linux `apt install nodejs`, und in MacOS manuell installieren.

2. **Website starten**:
Im Root-Ordner `npm start` ausführen.
